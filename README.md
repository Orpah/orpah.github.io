# 天逯
2001年参与设计二代证后台系统，我仿照神话传说中的【生死簿】，进行了部级系统设计。当时有些项目人员可能觉得建一个存十几亿人信息的大库很难，几年后，他们又觉得不难，十几年后，很简单。

通过二代证项目，中国的人口系统进入了2.0时代，1.0要追溯到公元585年，隋炀帝杨坚建立户籍制度。

现在咱们可以开始向3.0迈进，将人口数据在时间和空间维度进行延展，仿照【轮回系统】，做一个新的系统，名为【天逯】。

## 时间延展

以人口系统为基础，通过与其它系统和数据关联，会记录一个人样貌、身高、体重、DNA等各种信息。让这些信息为后世的个人发挥作用，即是天逯的目标。

例如，林徽因（1904年6月10日－1955年4月1日）这一世长相没变，一眼即可认出，大约1997-1999年出生。道家认为，前世与后世在相貌上变化很小。确信前世，对于她过好这一世会有很大帮助。基于长相来寻找自己的前世，是天逯系统的一个使用场景。

## 空间延展

### 10厘米
空间延展，是天逯系统的重点。在二代证项目里，端侧有二代证，二代证使用了近场通讯技术（基于ISO-14443协议），一个在10厘米范围内读取的无源证件。

### 1公里
天逯的第一步设想，是实现1公里范围、与无源端侧通讯，详见[本项目协议](https://github.com/Orpah/Protocol/blob/master/README_zh.md)。

WiFi Alliance 2016年推出了Halow协议，这个协议与天逯的第一步需求非常一致。2015年趁着WiFi Alliance在上海开会的机会，我在会议期间用1公里范围内寻人需求，简单忽悠了一下会议方，巧合的是，一年后，这个协议就推出了，并完美满足了本项目协议。

十年静待花开。现在淘宝上已经可以买到支持Halow协议的芯片了。我可以做天逯的演示系统了。

### 地球内
我奶奶1992年在獐子岛去世时，在我卧室墙上投出了她的脸，慈祥地望着我。北京与獐子岛的距离在几百公里。现在回想，这个投影，可能是基于量子的（光）。我觉得量子技术，能够实现地球内无源通讯。现在在淘宝上已经可以买到金刚石氮空位色心了，相信很快也能买到支持量子通讯的色心。
